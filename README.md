# SMSender

A SMS server written in Go (Golang).

## Install

```
go get -u github.com/minchao/smsender
```

## Brokers

Support brokers

* [AWS SNS (SMS)](https://aws.amazon.com/sns/)
* [Nexmo](https://www.nexmo.com/)
* [Twilio](https://www.twilio.com/)
